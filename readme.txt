This plugin helps you to manage and include ShareUsers links in your blog / Ayuda para gestionar e incluir los enlaces de ShareUsers en tu blog

== Description ==

This plugin helps users to include ShareUsers links in their blogs. [ShareUsers](http://shareusers.net/en/ "Share your webs and blogs users through ShareUsers") is a free online service which allows everyone to link up with other bloggers and webmasters.

Este plugin ayuda a los usuarios de ShareUsers a incluir los enlaces necesarios en sus blogs. [ShareUsers](http://shareusers.net/es/ "Comparte los usuarios de tus webs y blogs con ShareUsers") es un servicio gratuito que te permite interactuar con otros bloggers y webmasters.

== Installation ==

This section describes **how to install the plugin** and get it working:

1. Upload 'shareusers' directory to the '/wp-content/plugins/' folder in your blog
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your favourite options under 'Settings > ShareUsers'
4. Manage your keywords in the 'Tools > ShareUsers' menu

Esta seccion describe **como instalar el plugin** y hacer que funcione:

1. Sube el directorio 'shareusers' a la carpeta '/wp-content/plugins/' de tu blog
2. Activa el plugin desde el men&uacute; 'Plugins' de WordPress
3. Config&uacute;ralo a trav&eacute;s de 'Opciones > ShareUsers'
4. Gestiona tus palabras clave en el men&uacute; 'Herramientas > ShareUsers'

= What is ShareUsers? =

= What is the "Remote update frequence"? =

This setting allows you to establish how frequently you want the plugin to **connect to ShareUsers and download your subscriptions**.

= &iquest;Qu&eacute; es la "Frecuencia de actualizaci&oacute;n remota"? =

Esta opci&oacute;n te permite elegir cada cu&aacute;nto tiempo quieres que el plugin **se conecte a ShareUsers para descargar las suscripciones** de tu blog.

= What about 'Auto link' option? =

Check it if you want the plugin to **automatically link any occurrence of your subscribed keywords** in your blog.

= &iquest;Y la opcion 'Auto enlazado'? =

Marca esta casilla si quieres que el plugin **enlace autom&aacute;ticamente cualquier ocurrencia de las palabras clave** a las que est&aacute; suscrito tu blog.

= How can I check my blog's keywords? =

You can see your blog's subscriptions at [ShareUsers](http://shareusers.net/en "Share your webs and blogs users through ShareUsers") or through the 'Tools > ShareUsers' menu in WordPress.

= &iquest;C&oacute;mo puedo ver las palabras clave que est&aacute; usando mi blog? =
